新手小白搭建WordPress个人博客网站教程Windows版

Mr. J

云服务器建站

发布于：2020年7月18日

0  

本文面对所有建站新手，如果你对于建立个人站点无从下手，那么本文将会给你很大的帮助。本系列分为Windows版本和linux版本，指的是服务器的系统，如果你不知道该如何选择，建议是选择linux版本，linux版本进入控制台虽说是难控制一点，但安装了宝塔面板（一款服务器系统管理工具）的话，你就可以很轻松的安装任何环境。而且稳定性也比Windows更好一点。

本文先从Windows版本说起，如需查看linux版本，请访问——新手小白搭建WordPress个人博客网站教程Linux版

本文以腾讯云服务器为例

一、域名

建站，首先你需要一个域名。以腾讯云为例子，访问腾讯云官网域名注册，选择自己喜欢的 后缀，和自己的域名，然后下单即可，便宜的一年也就几块钱，具体看自己选择的域名咯。

如果不希望使用域名也可以跳过这一步。

因为首购优惠只有一次，所以建议域名放在最后再买，可以先买云服务器。

二、服务器

1，选择好域名后，开始选择服务器。这里以腾讯云为例。

如果你是学生或者你是第一次购买的话，是会有很大的优惠力度的
直达链接——【腾讯云官网服务器活动】

一般选择一款入门性能的服务器1核2G就足够个人博客使用了。

服务器的配置选择，活动地域选择离你最近的，操作系统选择Windows server 2008 或者 2012 都可以。地区选择离自己最近的。

选择好这些以后，其他只需要默认即可。

三、配置

下单以后，右上角信箱会显示有你刚刚买的服务器账号和密码，拿到账号和密码。

右下角选择云服务器会有服务器的IP地址，选择公网的IP地址就行。

按住键盘的电脑的win+R键（或者点击左下角开始，搜索运行）输入mstsc

回车进入远程连接页面

计算机填写你的服务器的公网IP地址
用户名一般是Administrator
密码查看上面获取到的账号密码。

进入服务器以后。安装xampp集成环境工具（可以在本地下载好以后，拖入服务器；也可以直接在服务器里面下载），工具从本页面下载，也可以从百度上下载站进行下载。

下载地址：下载

刚打开安装包的时候可能会有一个提示，ok跳过即可

然后一路next，默认不用修改

安装好以后，他会在你服务器的这个位置：

这里首先不要启动先，先依次配置一下

如果你发现主面板mysql 右边已经有数字，说明你的系统已经开启了对应的服务占用了当前的端口，请打开任务管理器，将mysqld.exe进程关掉。（因为使用集成工具比较好管理服务，所以建议关掉系统的，改用工具安装的Mysql服务。）

如果主面板右边没有对应的数字，请跳过这一步

关闭后：数字消失。

安装apache与Mysql服务，点击旁边的×号即可安装

如图即表示安装成功

然后依次点击右侧start按钮开启服务，如图即是开启成功

安装成功以后，环境这一部分已经解决。

四、安装WordPress程序

将WordPress程序包里面的所有文件粘贴在环境根目录下

程序包可以从网上下载：下载地址

返回我们的电脑，打开浏览器，输入http://xxx.xxx.xxx.xxx(服务器公有IP号):80

然后就会跳转到绑定数据库安装程序，到这里说明前面的步骤都没有问题。

五、数据库

接下来需要配置数据库的文件

首先创建一个数据库给WordPress使用

将数据库名设置成刚刚创建的
数据库用户名只能是root
数据库密码修改成为空！！！！！！

设置为空是因为，在Windows服务器下，记事本是有点毒的，内容编码不太一样，建议先留空，后续再进行修改。否则连接不上数据库的。

不要设置中文！！！

然后回到自己的电脑，点击现在开始安装

点击提交，看到如图页面，说明已经连接上数据库没有问题了。

然后填写网站的信息。

填写后，网站已经建成。

网站后台http:xxx.xxx.xxx.xxx/wp-admin/

网站地址http:xxx.xxx.xxx.xxx(你的公有IP)

六、域名绑定

在我们一开始就已经买好的域名，进入域名解析

国内大陆服务器需要备案

注意：国内的服务器绑定域名需要备案的哦，备案时长大概7个工作日。根据指示一步步上传站长资料即可。

然后等待十分钟，你就可以通过域名，以及www.域名访问你的网站了

免备案云服务器

如果你不想备案，想要直接就可以上线访问的话，可以选择你香港或者海外的云服务器，根据个人使用体验，腾讯云的香港云服务器比阿里云的香港服务器速度更快，体验更好。【点击进入腾讯云香港及海外云服务器优惠专区】

七、结束语

到此为止，在Windows操作系统上的WordPress建站程序已经安装完毕，你可以开始经营属于自己的网站了。

腾讯云服务器的配置如何选择？

Mr. J

云服务器建站

发布于：2020年7月17日

0  

云服务器的配置选择，和网站或应用的类型、访问量、数据量大小、程序质量等因素有关，建议和您的网站或应用的开发技术人员沟通，选择最适合您的配置。

如果您没有技术人员可提供建议，可以参考我们的建议进行配置选择。网站初始阶段访问量小，只需要一台低配置的服务器即可，应用程序、数据库、文件等所有资源均在一台服务器上。

腾讯云服务器具有强大的弹性扩展和快速开通能力。随着业务的增长，你可以随时在线增加服务器的CPU、内存、硬盘以及带宽等配置，或者增加服务器数量，无需担心低配服务器在业务突增时带来的资源不足问题，不让一个用户流失。

腾讯云服务器配置方案：

普通的个人小型网站，个人博客等小流量网站，可选择低配置的腾讯云服务器

推荐配置：CPU：1核、内存1G或2G、硬盘40G、带宽：1M或2M

论坛、门户类网站：论坛、门户类网站，用户活跃性与访问量较高，为了保证足够的服务器资源空间，提升访问速度。

推荐配置：CPU：2核、内存：4G、硬盘120G、带宽：3M

品牌官网类网站：对官网、品牌较为重视的政府、企业等， 为了保证网站浏览更加流利顺畅，提升政府、品牌形象。

推荐配置：CPU：4核、内存：8G、硬盘150G、带宽：4M

视频、购物类网站：视频、购物类网站，包含庞大的数据信息， 为了保证迅速的信息处理能力保证网站的点播、交易正常进行。

推荐配置：CPU：8核、内存：8G、硬盘300G、带宽：10M以上

游戏、软件类网站：对开发、测试、环境要求较高的游戏软件类网站，需要较高的资源配置带来更强劲的计算性能，保证业务需求。

推荐配置：CPU：16核、内存：16G、硬盘：500G、带宽：10M
